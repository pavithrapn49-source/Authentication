# User Authentication API

## Tech Stack
- Node.js
- Express.js
- MongoDB
- JWT
- Postman

## Features
- User Registration
- User Login
- JWT Token Authentication
- Protected Routes

## Installation

npm install
npm run dev

## Environment Variables

PORT=5000
MONGO_URI=your_mongodb_url
JWT_SECRET=your_secret_key

## API Endpoints

POST /api/auth/register
POST /api/auth/login
GET /api/user/me
